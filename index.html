Modals

Can
Can + Base Form - পারা
Can be + Noun/Adjective - হতে পারা
Can Have + Noun/Adjective - থাকতে পারা
Can not but + Base Form - না করে পারিনা
Can not let + obj + Base Form - করতে দিতে পারিনা
Could
Could + Base Form - পারতাম
Could be + Noun/Adjective - হতে পারত
Could have + Noun/Adjective - থাকতে পারতাম
Could not but + Base Form - না করে পারলাম না
Would
Would + Base Form - তাম
Would be + Noun/Adjective - হতাম
Would have + Noun/Adjective - থাকতো
Would like to + Base Form - চাওয়া
Would rather + Base Form - বরং করাই ভালো
Would you mind + V1 + ing - কিছু মনে না করলে
Should
Should + Base Form - উচিত
Should be + Noun/Adjective - হওয়া উচিত
Should have + Noun/Adjective - থাকা উচিত
Should be + V1 + ing - করতে থাকা উচিত
Should have + V3 - উচিত ছিল
May
May + Base Form - পারা
May be + Noun/Adjective - হতে পারা
May have + Noun/Adjective - থাকতে পারা
May be + V1 + ing - সম্ভবত করছে
May have + V3 - সম্ভবত করেছে
Might
Might + Base Form - করতেও পারা
Might be + Noun/Adjective - হতেও পারা
Might have + Noun/Adjective - থাকতেও পারা
Must
Must + Base Form - অবশ্যই
Must be + Noun/Adjective - অবশ্যই হব
Must have + Noun/Adjective - অবশ্যই থাকবে
Must have + V3 - অবশ্যই করে থাকবো/অবশ্যই ছিল
Will
Will + Base Form - হবে
Will be + Noun/Adjective - হতে হবে
Will have + Noun/Adjective - থাকতে হবে
Will have to + Base Form - যেতেই হবে
Need To
Need to + Base Form - দরকার/প্রয়োজন
Need to be + Noun/Adjective - হওয়া দরকার
Need to have + Noun/Adjective - থাকা দরকার
Need not have + V3 - না করলেও চলতো
Dare
Dare + Base Form - সাহস
Used To
Used to + Base Form - করতাম
Be (am,is,are) Used to + V + ing - অভ্যস্ত
Additional Forms
Be Supposed to + Base Form - কথা/কথা আছে
Be bound to + Base Form - বাধ্য
Am, Is, Are, Was, Were
Am to + Base Form - হয়
Is to + Base Form - হয়
Are to + Base Form - হয়
Was to + Base Form - হয়েছিল
Were to + Base Form - হয়েছিল
Have to + Base Form - হবে
Has to + Base Form - হবে
Had to + Base Form - হতো
Will have to + Base Form - যেতেই হবে
